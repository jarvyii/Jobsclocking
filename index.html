Timecards System

TimesCards System is a coorporate tool that provides a timesheet system with project organization features and time tracking. This software is WEB-based, and can be run on one PC, across a network or in any kinf of movil device. Project budgeting, reporting, and billing are easy to do with this software.

With the System you can have the control of the Timesheet of your employee, week per week and with different way of payments.

Options

MENUE:


Home: Allow you to use and update the timesheet

About: Give you all information about the System

Report: You can access to diffrent report in correspondt with access level

HH-HH: Menue use only for HH-RR Deparment. They can access to closed period, add emplyee and remove employee

Period to Search: Allow you to type and old and closed period. You can see it and print it.

TIMESHEET Fileds:

Name: First and Last name of the employee

From: The First day and the Last day of the week.

Date: The date of the month.

Description: Differents way of payments.

Timesheet: The employee have to update his work time, for Monday to Sunday. He has until Moday 10:00AM of the next week to update it.

Submit: To submit the the week work time. After it. The employee caný do any change. To do any change, the employee has to contact the supervisor or the HH-RR Department.

Save: Save last change to the system.

Restore: Restore the old vlaue saved in the Database.

Print: Print the work report.
